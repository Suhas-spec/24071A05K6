# ServletUserDetailsApp

Simple Java Servlet project with:

- `src/UserServlet.java`
- `WebContent/index.html`
- `WebContent/WEB-INF/web.xml`

## What It Does

- Shows a user form
- Accepts `name`, `email`, and `age`
- Submits the form to `UserServlet`
- Displays the entered details in a table

## Project Structure

```text
ServletUserDetailsApp/
├── src/
│   └── UserServlet.java
└── WebContent/
    ├── index.html
    └── WEB-INF/
        └── web.xml
```
