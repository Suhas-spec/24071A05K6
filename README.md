# Repository Projects

This repository contains only these two application projects:

- `hospital-management-system/` - main React hospital management application
- `UserServer/` - Java servlet-based user details application

## Main Hospital Application

The main hospital application source code is inside:

```text
hospital-management-system/
```

Run it locally with:

```bash
cd hospital-management-system
npm install
npm start
```

## UserServer Application

The servlet application source code is inside:

```text
UserServer/
```
